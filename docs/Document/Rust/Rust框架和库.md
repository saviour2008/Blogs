## 库搜索

crates：https://crates.io/

Lib.rs：https://lib.rs/

Rust的标准库非常精简，因此在实际开发中，我们通常需要依赖大量的第三方库。crates.io是官方的包托管网站，您可以在这里找到各种各样的Rust库。而docs.rs则是官方的文档托管网站，自动生成了所有发布在crates.io上的包文档。通过这两个网站，您可以轻松找到所需的库，并查看其详细文档。

### Slint

Slint 是一个声明性 GUI 工具包，用于为 Rust、C++ 或 JavaScript 应用程序构建本机用户界面。
